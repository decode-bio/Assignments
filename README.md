# # Assignment_1 (For module 2 & 3)
* Problem statement: Analyze category-wise performance and time-based trends using bar and line charts.
*
  **Tasks**
* Connect Tableau to the CSV/Excel data source.
* Create a Bar Chart showing sales or profit by category.
* Create a Time Series Chart (line chart) showing trends over time.
* Add filters to enable interactivity (e.g., region, category, time range).
* Customize labels, colors, and tooltips for clarity.
